# Job Search Command Map

Canonical command intents:

- `/score-companies` → run company scoring and reorder target list
- `/target-quality` → run target data hygiene checks (dedupe/stale/broken links/missing fields)
- `/research-company <name>` → create one-company dossier
- `/track add|update|list|followup|report` → application pipeline operations

Natural-language requests should resolve to one of these intents.
