# Career Manager Command Map

Canonical command intents:

- `/onboarding` -- first-time setup or profile update
- `/job-search` -- run the full pipeline (discovery + scoring + action list)
- `/research <company>` -- create one-company dossier
- `/tailor-cv <role at company>` -- tailor resume + cover letter
- `/tracker add|update|list|followup|report` -- application pipeline operations
- `/dashboard` -- open or regenerate the HTML dashboard

Natural-language equivalents:
- "What should I do today?" / "briefing" / "status" -- router runs briefing and suggests next action
- "Pursue [company]" -- research + tailor + track (sequential)
- "Find new companies" -- job-search pipeline
- "Set me up" -- onboarding
